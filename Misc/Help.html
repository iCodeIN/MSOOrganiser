Useful information

You can find the current development schedule here - ask Julia for access. All errors in the Organiser application are logged and (assuming an internet connection) reported to Julia automatically. Updates are delivered as and when ready - you are advised to keep the application up to date.

Web Links

The following pages are intended for retrieving reports on completed Olympiads. The "current Olympiad" means the most recent one in the database, which at the time of writing is 2016. Specific parameters can be varied as required.
  • http://mso.juliahayward.com/Report/Nationality shows the numbers of contestants for each nationality, in the current Olympiad
  • http://mso.juliahayward.com/Report/IndividualMedals shows the individual all-time medals table
  • http://mso.juliahayward.com/Report/ContestantMedals?contestantId=10392 shows the all-time record of a specific contestant - medals and titles.
  • http://mso.juliahayward.com/Report/YearMedals?year=2015 shows the medals table for a specific year, broken down by events. Parameter year is not optional.
  • http://mso.juliahayward.com/Report/GameMedals?gamecode=BA shows the medals table for a specific game, broken down by year and event.
  • http://mso.juliahayward.com/Report/PentamindStandings?year=2015 shows the Pentamind standings for a specific Olympiad. If parameter year is not supplied, then the current Olympiad is used.
The following are intended for live data for an ongoing Olympiad.
  • http://mso.juliahayward.com/Event/Entrants?year=2014&eventCode=BAOC shows the entrants for a given event. Parameter year is optional, if not supplied then uses the current Olympiad.
